    \subsection{\texorpdfstring{6. \emph{Using Pandas to read a column as a
Series}}{6. Using Pandas to read a column as a Series}}\label{using-pandas-to-read-a-column-as-a-series}

\paragraph{A pandas Series is very similar to a one-dimensional NumPy
array, but it has additional functionality that allows values in the
Series to be indexed using
labels.}\label{a-pandas-series-is-very-similar-to-a-one-dimensional-numpy-array-but-it-has-additional-functionality-that-allows-values-in-the-series-to-be-indexed-using-labels.}

\paragraph{A NumPy array does not have the flexibility to do this. This
labeling is useful when you are storing pieces of data that have other
data associated with
them.}\label{a-numpy-array-does-not-have-the-flexibility-to-do-this.-this-labeling-is-useful-when-you-are-storing-pieces-of-data-that-have-other-data-associated-with-them.}

\paragraph{Say you want to store the ages of students in an online
course to eventually figure out the average student
age.}\label{say-you-want-to-store-the-ages-of-students-in-an-online-course-to-eventually-figure-out-the-average-student-age.}

\paragraph{If stored in a NumPy array, you could only access these ages
with the internal ndarray indices 0,1,2\ldots. With a Series object, the
indices of values are set to 0,1,2\ldots{} by default, but you can
customize the indices to be other values such as student names so an age
can be accessed using a
name.}\label{if-stored-in-a-numpy-array-you-could-only-access-these-ages-with-the-internal-ndarray-indices-012.-with-a-series-object-the-indices-of-values-are-set-to-012-by-default-but-you-can-customize-the-indices-to-be-other-values-such-as-student-names-so-an-age-can-be-accessed-using-a-name.}
